The Utexo SDK provides programmatic access to the Utexo execution layer, enabling applications to send and receive native USDT on Bitcoin through RGB and Lightning without operating nodes, managing channels, or handling protocol-level infrastructure.

It exposes a clean set of async methods organised around wallet management, asset operations, Lightning payments, and on-chain interactions. All operations are performed locally using client-side validation — no shared state is sent to a central server during normal execution.

## What the SDK Provides

- **Wallet management** — key generation, derivation, initialisation, backup, and restore (including VSS cloud backup)
- **RGB asset operations** — asset issuance, blind invoices, witness invoices, asset transfers, and balance queries
- **Lightning payments** — invoice creation, payment execution (begin → sign → end flow), and payment status queries
- **On-chain interactions** — deposit address generation, BTC balance queries, on-chain withdrawal
- **UTXO management** — UTXO creation, listing, and state sync

## Execution Model

The SDK uses the `@utexo/rgb-sdk` package. The primary entry point is `UTEXOWallet`, which is initialised with a mnemonic and optional `{ network, dataDir }` configuration.

Key execution properties:

- All operations are **non-custodial** — the SDK never transmits private keys or mnemonics
- All API calls execute with **predefined costs and latency** — no fee auction or gas estimation required
- The SDK is **Node.js only** — it is not browser-compatible
- Payments follow a **begin → sign → end** flow for Lightning and a **begin → sign → end** flow for on-chain withdrawals, separating unsigned PSBT generation from signing

## Networks

The SDK supports three environments:

| Environment | RGB Transport | Bitcoin Indexer |
|---|---|---|
| Mainnet | `rpcs://rgb-proxy-mainnet.utexo.com/json-rpc` | `ssl://electrum.iriswallet.com:50003` |
| Testnet | `rpcs://rgb-proxy-testnet3.utexo.com/json-rpc` | `ssl://electrum.iriswallet.com:50013` |
| Signet | `rpcs://rgb-proxy-utexo.utexo.com/json-rpc` | `https://esplora-api.utexo.com` |
