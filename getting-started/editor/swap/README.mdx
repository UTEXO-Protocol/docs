# Swap

**Overview**

The Utexo Swap Dex ensures non-custodial internal BTC↔USDT execution for settlement, treasury and payments, reducing reliance on centralized venues for rebalancing and conversion.

It enables native USDT ↔ BTC swaps on Bitcoin, privacy-preserving swaps to prevent front-running, and LP functionality for RGB-native assets. Trades are settled with instant finality and predictable fees.
