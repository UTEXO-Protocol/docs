## Your First Integration

This guide walks you through installing the Utexo SDK, initializing a wallet, and completing your first USDT transfer on Bitcoin — from zero to working code.

Estimated time: 15 minutes.

## Prerequisites

Before you start:

- Node.js v18 or later installed
- A testnet BTC balance for transaction fees (use a testnet faucet)
- Basic familiarity with async/await JavaScript

## Step 1: Install the SDK

```bash
npm install @utexo/rgb-sdk
```

The SDK is Node.js only and is not browser-compatible.

## Step 2: Generate Wallet Keys

```javascript
const { generateKeys } = require('@utexo/rgb-sdk');

const keys = await generateKeys('testnet');
console.log('Mnemonic:', keys.mnemonic); // Store this securely
```

This generates a BIP-39 mnemonic. Store it securely — it is the only way to recover your wallet.

## Step 3: Initialize the Wallet

```javascript
const { UTEXOWallet } = require('@utexo/rgb-sdk');

const wallet = new UTEXOWallet(keys.mnemonic, { network: 'testnet' });
await wallet.initialize();
```

Initialization connects to the default RGB transport and Bitcoin indexer endpoints for testnet.

## Step 4: Get a Deposit Address

```javascript
const address = await wallet.getAddress();
console.log('Deposit address:', address);
```

Use this address to receive BTC on testnet. You need a small BTC balance to cover RGB transaction fees.

## Step 5: Create UTXOs

Before receiving RGB assets, the wallet needs UTXOs to anchor RGB state.

```javascript
await wallet.createUtxos({ num: 5, size: 1000 });
await wallet.refreshWallet();
```

## Step 6: Generate an RGB Invoice

To receive USDT (or any RGB asset), generate a blinded invoice:

```javascript
const receiveData = await wallet.blindReceive({
  assetId: 'your-rgb-usdt-asset-id',
  amount: 100,
  minConfirmations: 1,
  durationSeconds: 3600
});
console.log('RGB Invoice:', receiveData.invoice);
```

Share this invoice with the sender. It expires after `durationSeconds`.

## Step 7: Send an RGB Asset

Once you have an RGB asset balance, send it to another wallet:

```javascript
await senderWallet.send({
  invoice: receiveData.invoice,
  assetId: 'your-rgb-usdt-asset-id',
  amount: 100
});

await senderWallet.refreshWallet();
await receiverWallet.refreshWallet();
```

## Step 8: Check Balances

```javascript
const assets = await wallet.listAssets();
console.log('Assets:', assets);

const btcBalance = await wallet.getBtcBalance();
console.log('BTC balance:', btcBalance);
```

## Default Network Endpoints

The SDK connects to the following endpoints by default on testnet:

- **RGB Transport:** `rpcs://rgb-proxy-testnet3.utexo.com/json-rpc`
- **Bitcoin Indexer:** `ssl://electrum.iriswallet.com:50013`

For mainnet, see the [SDK reference](/sdk/utexo-sdk) for mainnet endpoint configuration.
